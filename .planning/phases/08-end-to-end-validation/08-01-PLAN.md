**Step 1: Create `tests/helpers.ts`** with shared test utilities:

- `callChatApi(options: { messages, phase?, phase3_context?, baseUrl? })` -- wraps `fetch` POST to `${baseUrl}/api/chat` (default `http://localhost:5173`). Sends JSON body with `messages`, optional `phase` and `phase3_context`. Include auth header if `TEST_AUTH_TOKEN` env var is set: `Authorization: Bearer ${process.env.TEST_AUTH_TOKEN}`. Returns `{ status, body }` where body is parsed JSON.
- `skipIfNoApi()` -- checks if the dev server is reachable by fetching the base URL with a 3-second timeout. If not reachable, calls `test.skip()` with message "Dev server not running at {baseUrl}". Per D-08, tests skip gracefully when infrastructure is unavailable.
- `delay(ms: number)` -- simple `new Promise(resolve => setTimeout(resolve, ms))` for rate limiting between calls.
- Re-export `parseScoresFromResponse` and `stripScoresFromResponse` from `../src/lib/score-parser` so tests can import from one place.
